# GitHub Keyboard Shortcuts & Commands

| Category | Shortcut / Command | Action |
|:---|:---|:---|
| Global | `?` | Show keyboard shortcuts help |
| Global | `Ctrl + K` or `Ctrl + Shift + P` | Open Command Palette |
| Global | `Esc` | Close modals, dropdowns, or dismiss |
| Global | `s` or `/` | Focus search bar |
| Global | `Shift + ?` | Open keyboard shortcuts (when not in text field) |
| Navigation | `g` then `d` | Go to Dashboard |
| Navigation | `g` then `c` | Go to Code tab |
| Navigation | `g` then `i` | Go to Issues |
| Navigation | `g` then `p` | Go to Pull Requests |
| Navigation | `g` then `a` | Go to Actions |
| Navigation | `g` then `b` | Go to Projects |
| Navigation | `g` then `w` | Go to Wiki |
| Navigation | `g` then `s` | Go to Settings |
| Navigation | `g` then `r` | Go to Repository main page |
| Navigation | `g` then `n` | Go to Notifications |
| Navigation | `g` then `o` | Go to Profile |
| Navigation | `g` then `u` | Go to your repositories |
| Navigation | `g` then `m` | Go to your organizations |
| Navigation | `g` then `y` | Go to your projects |
| Navigation | `g` then `t` | Go to Topics |
| Repository | `.` | Open repository in github.dev (web editor) |
| Repository | `,` | Open repository in GitHub Codespaces |
| Repository | `t` | Activate File Finder |
| Repository | `l` | Jump to line in code |
| Repository | `w` | Open branch/tag selector |
| Repository | `y` | Convert URL to permanent link (permalink) |
| Repository | `i` | Show/hide comments on diffs |
| Repository | `a` | Show/hide annotations on diffs |
| Repository | `b` | Blame view (on code page) |
| Issues & PRs | `c` | Create new issue |
| Issues & PRs | `Ctrl + .` or `Cmd + .` | Open saved replies |
| Issues & PRs | `r` | Reply (quoting selected text) |
| Issues & PRs | `q` | Request reviewer's attention |
| Issues & PRs | `a` | Request author's attention |
| Issues & PRs | `Shift + r` | Mark notification as read |
| Issues & PRs | `Shift + m` | Mark notification as unread |
| Issues & PRs | `Shift + i` | Mark notification as done |
| Issues & PRs | `e` | Edit comment / issue / PR |
| Issues & PRs | `j` | Move selection down (issue/PR list) |
| Issues & PRs | `k` | Move selection up (issue/PR list) |
| Issues & PRs | `o` or `Enter` | Open selected issue/PR |
| Issues & PRs | `x` | Select/deselect item (with checkbox) |
| Issues & PRs | `Shift + c` | Close selected issue/PR |
| Issues & PRs | `Shift + o` | Reopen selected issue/PR |
| Issues & PRs | `Shift + l` | Add label to selected issue/PR |
| Issues & PRs | `Shift + a` | Assign selected issue/PR |
| Issues & PRs | `Shift + m` | Milestone selected issue/PR |
| Issues & PRs | `Shift + p` | Set project for selected issue/PR |
| Issues & PRs | `Shift + n` | Add notification subscription |
| Issues & PRs | `Space` | Preview issue/PR in hovercard |
| Notifications | `j` | Move down in notifications list |
| Notifications | `k` | Move up in notifications list |
| Notifications | `o` or `Enter` | Open selected notification |
| Notifications | `e` | Archive notification |
| Notifications | `Shift + u` | Mark notification as unread |
| Notifications | `Shift + m` | Mute thread |
| Notifications | `Shift + i` | Mark as done |
| Notifications | `Shift + r` | Mark as read |
| Code Review | `j` | Move to next diff |
| Code Review | `k` | Move to previous diff |
| Code Review | `n` | Move to next comment |
| Code Review | `p` | Move to previous comment |
| Code Review | `Shift + n` | Move to next file in PR |
| Code Review | `Shift + p` | Move to previous file in PR |
| Code Review | `r` | Start review |
| Code Review | `Ctrl + Shift + Space` | Add single comment |
| Code Review | `q` | Request changes |
| Code Review | `a` | Approve PR |
| Command Palette | `Ctrl + K` then `Ctrl + C` | Copy URL to clipboard |
| Command Palette | `Ctrl + K` then `Ctrl + O` | Open in github.dev |
| Command Palette | `Ctrl + K` then `Ctrl + S` | Open in codespace |
| Command Palette | `Ctrl + K` then `Ctrl + G` | Copy GitHub link as markdown |
| Command Palette | `Ctrl + K` then `Ctrl + R` | Copy repository path |

---

*Note: Most shortcuts work when you are not actively typing in a text field. Press `?` on any GitHub page to see available shortcuts for that specific page.*
